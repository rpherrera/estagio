\begin{abstract}

\doublespacing {

    This work exposes one of the actual greatest treats for the servers which are based on \textit{Unix} families, the brute force attacks wich uses dictionaries against the \textit{Secure Shell} (\textit{SSH}) service. With such premisse, were developed a \textit{Web} tool which can identify the attack origins, generate reports and rules that can be used in Firewalls to filter these treats.

    \textbf {
        Keywords: Security, Unix, Firewall, SSH, Brute Force.
    }
}
