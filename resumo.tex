\begin{resumo}

\doublespacing {

    Este trabalho expõe uma das grandes ameaças atuais aos servidores baseados nas famílias Unix, os ataques de força bruta que utilizam dicionários contra o serviço \textit{Secure Shell} (\textit{SSH}). Com tal premissa, foi desenvolvida uma ferramenta \textit{Web} capaz de identificar as origens dos ataques, gerar relatórios e regras que podem ser utilizadas em Firewalls para o filtro destas ameaças.

    \textbf {
        Palavras-chave: Segurança, Unix, Firewall, SSH, Força Bruta.
    }
}

\end{resumo}
